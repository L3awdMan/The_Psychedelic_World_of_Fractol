<h2 >INSTALLATION </h2>

<br>

```bash
# Clone the repository
git clone https://github.com/yourusername/fractol.git

# Navigate to directory
cd fractol

# Compile
make

# Run
./fractol mandelbrot
./fractol julia -0.7 0.27015
./fractol newton
./fractol flower
```
